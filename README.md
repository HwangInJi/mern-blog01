# MERN BLOG

- vite(https://ko.vitejs.dev/guide/)
- tailwind(https://tailwindcss.com/docs/guides/vite)

```bash
npm create vite@latest
cd client
npm i
npm run dev
```

## client

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
npm i react-router-dom
npm i react-icons
```
